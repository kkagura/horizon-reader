# AI 新闻日报

这是本地开发示例。服务器部署后，将 `REPORTS_DIR` 指向 Horizon 生成的 Markdown 目录：

```text
/opt/horizon-deploy/reports
```

## 今日重点

- Horizon 已经完成新闻采集和 AI 筛选。
- 阅读器服务会在运行时读取 Markdown 文件。
- 新增日报文件后不需要重新构建 Nuxt 应用。
